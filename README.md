# Callnest Landing Page

Strona internetowa Callnest zbudowana w React + Vite + TypeScript na podstawie projektu z folderu `Landing page`.

## Uruchomienie

```bash
cd callnest-landing
npm install
npm run dev
```

Strona będzie dostępna pod adresem `http://localhost:5173`.

## Build produkcyjny

```bash
npm run build
npm run preview
```

## Sekcje

- Nawigacja z efektem blur przy scrollu
- Hero z animowanym equalizerem
- Statystyki z animacją licznika
- Jak to działa (4 kroki)
- Zastosowania (3 karty)
- Branże (tagi)
- Dlaczego Callnest
- Cennik (4 pakiety)
- CTA i stopka
